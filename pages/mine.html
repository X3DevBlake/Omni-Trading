📱 Phone Mining Guide

▼
1

Create Your Wallet (already done if you see your address above!)

Go to the Wallet Creator and generate your secure Omni wallet. Your key never leaves the browser.

→ Create Wallet
2

Start Browser Mining (This Page!)

Tap the big mining button above. Mining uses Web Worker threads running real SHA-256 Proof-of-Work directly in your browser. Works on iPhone and Android.

3

Add to Home Screen (Optional but Recommended)

iPhone (Safari): Tap Share → "Add to Home Screen"
Android (Chrome): Tap ⋮ → "Add to Home Screen"
This makes it feel like a native app and keeps the screen on.

4

Advanced: Mine on Android with Python (Termux)

For maximum hashrate, install Termux from F-Droid and run the actual Python mining engine:

pkg update && pkg install python -y pip install colorama cryptography

Then copy your mining script to Termux and run:

python mine_omni.py --threads 2
5

Keep Your Screen On

Mining pauses when the browser suspends. Go to your phone's Display Settings and disable auto-lock while mining.

iPhone: Settings → Display & Brightness → Auto-Lock → Never
Android: Settings → Display → Screen timeout → Never (or longest)

6

View Your Wallet & Earnings

Your mined OMNI accumulates in your wallet. Import to MetaMask with Chain ID 8888 and RPC http://127.0.0.1:8545 when running the full node.

→ Open Wallet Dashboard
